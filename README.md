# AI-Based Rock, Paper, Scissors Game

## Description
This is a Python-based Rock, Paper, Scissors game where the computer uses simple AI logic to predict the user's next move.

## Features
- AI prediction logic
- Score tracking
- Interactive gameplay
- User vs Computer

## Technologies Used
- Python
- Google Colab

## How to Run
1. Run the Python file
2. Enter rock, paper, or scissors
3. Type 'exit' to stop the game
